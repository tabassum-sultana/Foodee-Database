Presentation-e short explanation.

Database connected? Yes, Node.js backend connects to MySQL database named foodee_db.

Tables created? Yes, server automatically creates customer, order, cart, wishlist/activity, admin, and product tables.

Data save hoy kothay? Customer login saves in customers, add-to-cart/wishlist saves in cart_events, order saves in orders and order_items.
